Learn Everythings About Mobile Phones.

6 Lessons / Week

Course Description

  • STAGE 1:

    • How to lock/unlock a cell phone i.e turning on/off the mobile phone.

  • STAGE 2:

    • How to insert sim cards, memory cards, battery(if allowed manually) in a smart phone.

  • STAGE 3:

    • How to handle the touch screen feature of smart phones, and getting acquainted with it.

  • STAGE 4:

    • Understanding the methods of how to make a call and how to recieve it,cut it etc.

  • STAGE 5:

    • How to install or uninstall any application on the mobile phone.

  • STAGE 6:

    • How to open any search result after searching in web browser.

  • STAGE 7:

    • How to run audio or video files on the phone.

  • STAGE 8:

    • How to connect to internet and to make a gmail account.

  • STAGE 9:

    • Enable user to open one's gmail account & then making user understand how to type and send an email.

  • STAGE 10:

    • Making user able to search videos on youtube and familiarizing user with various option of youtube.

  • STAGE 11:

    • How to make a video call and switch between multiple sim cards in a cell phone.

  • STAGE 12:

    • Understanding the various settings required for phone required functionalities.

  • STAGE 13:

    • How to create accounts on several apps and perform some tinkering in order to be fully familiar.

  • STAGE 14:

    • How to recharge sim and other services on the mobile phone.

  • STAGE 15:

    • How to enable safety and security settings on the mobile phones.
